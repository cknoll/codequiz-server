About this project

CodeQuiz is a web platform for source code related quiz-tests. Currently it is used in one specific course (Python for Engineers), held in two universities [1].

The web application is implemented in Python using the django framework. It is still under active development, as user comments reveal bugs and suggest features. The project is open source and anybody is encouraged to use it and/or to contribute [2].

[1] http://tu-dresden.de/pythonkurs

[2] https://github.com/cknoll/codequiz-server
